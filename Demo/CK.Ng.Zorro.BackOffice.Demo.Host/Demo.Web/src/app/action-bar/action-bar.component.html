This component is an action bar. It takes a typed ActionBarContent.
It is used in the Layout component but you can use it as a standalone component as well.

You can specify an IconDefinition from FontAwesomeModule on each Action button.

Each icon has a margin-right that you can modify by assigning @ck-action-bar-button-margin variable in your global less stylesheet.

Each button holds ck-backoffice-action-bar-button class.
